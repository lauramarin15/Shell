◆ leer_linea()

char * leer_linea ( void )

Lee una línea de la entrada estándar.

Returns
char* Puntero a la cadena leída (debe ser liberada con free).

Lee una línea de la entrada estándar.

Utiliza la función getline(), que es una extensión segura de GNU C library. getline() maneja automáticamente la asignación de memoria: si la línea es demasiado larga para el buffer, getline() lo redimensiona usando realloc().

Returns
char* Puntero a la cadena de caracteres (string) leída. IMPORTANTE: El llamador es responsable de liberar (free) esta memoria.

◆ loop_shell()

void loop_shell ( )

Inicia el bucle principal de la shell. No retorna hasta que el comando 'salir' sea invocado.

Inicia el bucle principal de la shell. No retorna hasta que el comando 'salir' sea invocado.

Ciclo de vida:

  1. Read: Muestra el prompt "EAFITos>" y lee la entrada.
  2. Parse (Eval implícito): Divide la entrada en argumentos.
  3. Execute: Busca y corre el comando.
  4. Loop: Repite o libera memoria.

Parsea una línea cruda en un arreglo de tokens.

Esta función implementa el análisis léxico básico. Transforma: "calc 10 + 20" ---> ["calc", "10", "+", "20", NULL]

Utiliza memoria dinámica porque no sabemos a priori cuántos argumentos introducirá el usuario.

Parameters
lineaLa cadena de texto cruda leída anteriormente.
Returns
char** Un arreglo de cadenas (doble puntero) terminado en NULL.
